🔌 Wazuh USB Block

Enterprise USB blocking system deployed across Windows, Linux & macOS via Wazuh SIEM agent groups. Detects unauthorized USB storage in under 0.5 seconds and auto-ejects — with full JSON audit trail and zero manual steps on endpoints.

  • 3 policy groups: Block · Exception · Revoke
  • Whitelist by Vendor ID + Product ID + Serial Number
  • macOS: LaunchDaemon + log stream (event-driven, like udev)
  • Linux: udev rules + systemd service
  • Windows: Registry policy + PowerShell + Scheduled Task
  • Every event logged to JSON for Wazuh SIEM dashboards

🔐 HexaVault — AI-Powered VAPT Platform

Full-stack security audit platform that orchestrates 20+ scanning tools (Nmap, Nuclei, Nikto, OpenVAS, SQLMap, Subfinder and more) through a unified dashboard. Features an AI Co-Pilot for chat-based scan control, automated exploit validation (SQLi, XSS, SSRF, IDOR, CORS), attack path detection, pentest memory, and enterprise PDF reports with OWASP/PCI DSS/NIST/ISO 27001 compliance mapping.
