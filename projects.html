🔌 Wazuh USB Block — Enterprise Endpoint Security

Automated USB storage blocking deployed across Windows, Linux & macOS via Wazuh SIEM agent groups — zero manual steps on endpoints. Unauthorized devices are detected and ejected in under 0.5 seconds, with full JSON audit trail sent to Wazuh for SOC analysis.

🚫 Block Group
All USB blocked except IT-whitelisted devices (by VID + PID + Serial)
⚙️ Exception Group
Per-user approved device list — specific users get their devices whitelisted
🔓 Revoke Group
One Wazuh group change removes all USB restrictions — logs kept for audit
📊 Full Observability
Every plug-in event logged as JSON — ready for Wazuh dashboards & alerts

Full-stack security audit platform that orchestrates 20+ scanning tools (Nmap, Nuclei, Nikto, OpenVAS, SQLMap, Subfinder and more) through a unified dashboard.
