About Me

I'm a Security Analyst and VAPT Specialist focused on offensive security, blue-team operations, and security automation. I find what's broken — then I build tools so it doesn't break again.

My work spans web, network, API and cloud penetration testing, SOC engineering with Wazuh SIEM, threat hunting, and incident response. When a workflow is repetitive, I automate it end-to-end.

I've shipped a full AI-powered VAPT platform (HexaVault), an automated vulnerability patch pipeline, USB-control and endpoint-monitoring tooling — all built for real-world cyber defense.
